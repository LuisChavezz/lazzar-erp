
## Instrucciones para correr el proyecto

### 1. Instalar dependencias

Ejecuta uno de los siguientes comandos en la terminal para instalar las librerías necesarias:

```bash
npm install
# o
yarn install
# o
pnpm install
# o
bun install
```

### 2. Iniciar el servidor de desarrollo

Una vez instaladas las dependencias, inicia el servidor local:

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver la aplicación.

### 3. Build para producción (Opcional)

Para construir y correr la versión optimizada para producción:

```bash
npm run build
npm start
```
